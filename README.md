## About

A feature-rich admin panel dashboard for managing users, content, and viewing analytics. Built with Next.js and TypeScript for a responsive admin experience.

## Features

- 👥 **User Management** — CRUD operations for users
- 📊 **Analytics Dashboard** — Charts and statistics
- 🔐 **Role-based Access** — Admin and user roles
- 📋 **Data Tables** — Sortable, filterable tables
- 🎨 **Modern UI** — Clean admin interface

## Getting Started

```bash
git clone https://github.com/foxnaim/Trial-Admin-Ponel.git
cd Trial-Admin-Ponel
yarn install
yarn dev
```
